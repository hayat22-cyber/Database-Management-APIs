 Month 3: Database Management & APIs

This project focuses on building a basic data engineering pipeline using Python.
The objective is to understand how data is stored, processed, and retrieved using
databases and APIs.

Project: Weather Data Pipeline System

In this project, a complete workflow was created to manage weather-related data.

Key Tasks Performed
 Loaded provided CSV data into Python using pandas
 Designed and created an SQLite database
 Inserted structured data into database tables
 Integrated a weather API to fetch live weather data
 Stored API data into the database
 Performed data validation and basic error handling
 Queried stored data for analysis

Tools & Technologies Used
 Python
 Pandas
 SQLite
 Requests (API handling)
 Jupyter Notebook

Learning Outcomes
 Understanding database connections in Python
 Writing data to SQL databases using pandas
 Fetching and handling API data
 Building a simple ETL (Extract, Transform, Load) pipeline
 Managing structured data efficiently

This project demonstrates foundational skills in database management,
API integration, and data pipeline development.
